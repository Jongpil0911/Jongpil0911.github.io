This is an example post with some [plotly](https://plotly.com/javascript/) code.

````markdown
```plotly
{
  "data": [
    {
      "x": [1, 2, 3, 4],
      "y": [10, 15, 13, 17],
      "type": "scatter"
    },
    {
      "x": [1, 2, 3, 4],
      "y": [16, 5, 11, 9],
      "type": "scatter"
    }
  ]
}
```
````

Demo -> 3D plot

```plotly
{
  "data": [
    {
      "x": [1, 2, 3, 4],
      "y": [10, 15, 13, 17],
      "z": [5, 2, 8, 4], 
      "mode": "markers",
      "type": "scatter3d"
    }
  ],
  "layout": {
    "scene": { // 3D 그래프 레이아웃은 'scene' 객체 내부에 정의
      "xaxis": {"title": "X Axis"},
      "yaxis": {"title": "Y Axis"},
      "zaxis": {"title": "Z Axis"}
    }
  }
}
```
